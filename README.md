## Deploy

### Manually

``` bash
$ docker build -t philbot/youtube-comment-scraper2.0 .
$ docker run -p 49162:3000 -d philbot/youtube-comment-scraper2.0
```


### Docker Hub

``` bash
$ docker pull philbot/youtube-comment-scraper2.0
$ docker run -p 49162:3000 -d philbot/youtube-comment-scraper2.0
```
